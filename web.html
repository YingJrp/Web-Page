Wildfire Surveillance System Using LoRa Mesh Network

This Project aims to develop a wildfire prevention system utilizing a LoRa mesh network as a medium for wildfire monitoring and preparedness. The system seeks to reduce the need for manpower in wildfire detection by employing sensors to evaluate conditions that may lead to fires. When specific fire-prone conditions are detected, the system transmits data through microcontroller boards, which manage the information on a centralized server. Communication is based on a mesh networking model, where nodes that cannot directly transmit data to the gateway relay the information through the nearest nodes until it reaches the gateway.
